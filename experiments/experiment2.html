实验二：土的界限含水率试验

掌握土的液限、塑限测定方法，了解土的可塑性指标

一、实验目的

  1. 掌握土的液限、塑限测定方法
  2. 计算土的塑性指数和液性指数
  3. 确定土的分类和状态
  4. 为地基基础设计和施工提供必要的参数

二、实验原理

细粒土（黏性土）随着含水率的变化，会呈现出不同的物理状态：固态、半固态、可塑状态和流动状态。这些状态之间的分界含水率称为界限含水率。

2.1 界限含水率的定义

1. 液限（w_L）：土由可塑状态转变为流动状态时的界限含水率

2. 塑限（w_P）：土由半固态转变为可塑状态时的界限含水率

3. 缩限（w_S）：土由固态转变为半固态时的界限含水率

2.2 塑性指标

1. 塑性指数（I_P）：液限与塑限的差值，表示土的可塑性大小

I_P = w_L - w_P

2. 液性指数（I_L）：土的天然含水率与塑限的差值除以塑性指数，表示土的软硬状态

I_L = (w - w_P) / I_P

2.3 试验方法原理

液限试验（圆锥仪法）：将土样调成均匀的浓糊状，装入试杯中，刮平表面。然后将质量为76g的圆锥仪（锥角30°）在土样表面5秒内自由下落，记录圆锥入土深度。当圆锥入土深度为10mm时的土样含水率即为液限。

塑限试验（搓条法）：将土样调成土团，在毛玻璃板上用手掌搓成直径为3mm的土条，若土条在直径达到3mm时刚好断裂，则此时土样的含水率即为塑限。

三、实验设备

图1：界限含水率试验常用设备

3.1 液限试验设备（圆锥仪法）

  • 光电式液限仪：包括圆锥仪（质量76g，锥角30°）、电磁装置、显示器等
  • 试杯：直径不小于40mm，高度不小于50mm
  • 调土刀、刮土刀
  • 天平：感量0.01g
  • 烘箱、铝盒、干燥器
  • 蒸馏水瓶、凡士林

3.2 塑限试验设备（搓条法）

  • 毛玻璃板：尺寸约200mm×200mm
  • 天平：感量0.01g
  • 烘箱、铝盒、干燥器
  • 调土刀、凡士林
  • 蒸馏水瓶

四、实验步骤

图2：圆锥仪法测定液限

4.1 液限试验（圆锥仪法）

  1. 取代表性土样200g，过0.5mm筛，放入蒸发皿中
  2. 加入适量蒸馏水，用调土刀充分拌和，调成均匀的浓糊状
  3. 将制备好的土样分层装入试杯中，用力压实，使空气逸出
  4. 用刮土刀将土样表面刮平，使土样略高于试杯边缘
  5. 将试杯放在液限仪的升降座上，调整水平
  6. 提起圆锥仪，使圆锥尖接触土样表面（避免冲击）
  7. 按动电磁装置，使圆锥仪在5秒内自由下落
  8. 记录圆锥入土深度（h）
  9. 用调土刀在圆锥周围取出约10g土样，放入铝盒中，测定含水率
  10. 重复步骤3-9，制备不同含水率的土样，使圆锥入土深度分别接近10mm（液限）、7mm和13mm左右
  11. 绘制圆锥入土深度与含水率的关系曲线（h-w曲线），查得圆锥入土深度为10mm时的含水率即为液限
图3：搓条法测定塑限

4.2 塑限试验（搓条法）

  1. 取代表性土样100g，过0.5mm筛，放入蒸发皿中
  2. 加入适量蒸馏水，用调土刀充分拌和，使土样达到可塑状态
  3. 取一小块土样（约8-10g），用手捏成土团
  4. 将土团放在毛玻璃板上，用手掌轻轻搓揉，使土团逐渐成为土条
  5. 继续搓揉土条，直到土条直径达到3mm时刚好断裂
  6. 收集断裂的土条（约3-5g），放入铝盒中，测定含水率
  7. 重复步骤3-6，至少进行两次平行试验
  8. 取两次平行试验的平均值作为塑限

5.4 土的状态判定

液性指数I_L 土的状态
I_L ≤ 0 坚硬
0 ＜ I_L ≤ 0.25 硬塑
0.25 ＜ I_L ≤ 0.75 可塑
0.75 ＜ I_L ≤ 1.0 软塑
I_L ＞ 1.0 流塑

六、注意事项

  1. 液限试验时，土样应充分拌和，避免含有空气泡
  2. 圆锥仪应在5秒内自由下落，避免人为干扰
  3. 塑限试验时，搓条速度应均匀，用力适中
  4. 当土条直径接近3mm时，应注意观察断裂情况，避免过度搓揉
  5. 所有称量操作应迅速，避免水分蒸发影响试验精度
  6. 试验结束后，应及时清理试验设备，保持实验室整洁

七、思考题

  1. 为什么细粒土才有液限和塑限？粗粒土是否有界限含水率？
  2. 塑性指数的大小说明了什么？它对土的工程性质有什么影响？
  3. 液性指数如何反映土的软硬状态？在工程中有什么应用？
  4. 界限含水率试验的精度受哪些因素影响？如何提高试验精度？
  5. 不同成因、不同矿物成分的黏性土，其界限含水率有什么特点？
