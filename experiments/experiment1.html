实验一：土的基本物理性质试验

掌握土的密度、含水率、比重等基本物理指标的测定方法

一、实验目的

  1. 掌握土的密度、含水率、比重等基本物理指标的测定方法
  2. 学会计算土的孔隙比、孔隙率、饱和度等物理性质指标
  3. 了解土的三相组成及其相互关系
  4. 为土的工程分类和工程性质评价提供基本数据

二、实验原理

土的基本物理性质指标是反映土的物质组成和结构特性的重要参数。这些指标之间存在着一定的换算关系，可以通过三相草图进行推导。

2.1 土的三相组成

土是由固体颗粒、水和气体三相组成的体系。土的物理性质指标就是描述这三相物质在数量上和质量上的相对比例关系的参数。

基本物理指标：

1. 密度（ρ）：单位体积土的质量，单位为g/cm³或kg/m³

2. 含水率（w）：土中水的质量与固体颗粒质量的比值，以百分数表示

3. 土粒比重（Gₛ）：土粒的质量与同体积4℃时纯水质量的比值，无量纲

换算指标：

1. 干密度（ρd）：单位体积土中固体颗粒的质量，单位为g/cm³或kg/m³

2. 孔隙比（e）：土中孔隙体积与固体颗粒体积的比值，无量纲

3. 孔隙率（n）：土中孔隙体积与总体积的比值，以百分数表示

4. 饱和度（Sᵣ）：土中水的体积与孔隙体积的比值，以百分数表示

换算关系：

ρd = ρ / (1 + w)

e = (Gₛ × ρw × (1 + w)) / ρ - 1

n = e / (1 + e) × 100%

Sᵣ = (w × Gₛ) / e × 100%

式中：ρw为水的密度，取1g/cm³

三、实验设备

图1：土的基本物理性质试验常用设备

3.1 密度试验设备

  • 环刀：内径61.8mm，高20mm，容积约60cm³
  • 天平：感量0.1g
  • 切土刀、钢丝锯、凡士林
  • 玻璃板、秒表

3.2 含水率试验设备

  • 烘箱：能保持温度105-110℃
  • 天平：感量0.01g
  • 铝盒、干燥器
  • 坩埚钳

3.3 比重试验设备

  • 比重瓶：容量100mL或50mL
  • 天平：感量0.001g
  • 恒温水槽：能保持温度20±1℃
  • 真空抽气设备
  • 烘箱、干燥器
  • 温度计：精度0.5℃

四、实验步骤

图2：环刀法测定土的密度

4.1 密度试验（环刀法）

  1. 用天平称取环刀的质量（m₁），准确至0.1g
  2. 在环刀内壁涂一薄层凡士林，以减小切土时的摩擦
  3. 将环刀刃口向下，放在土样上，用切土刀将土样削成略大于环刀的土柱
  4. 将环刀垂直下压，边压边削，直至土样完全充满环刀
  5. 用切土刀将环刀两端多余的土削平，擦净环刀外壁
  6. 用天平称取环刀和土样的总质量（m₂），准确至0.1g
  7. 计算土的密度：ρ = (m₂ - m₁) / V，其中V为环刀的容积

图3：烘干法测定土的含水率

4.2 含水率试验（烘干法）

  1. 用天平称取铝盒的质量（m₀），准确至0.01g
  2. 取代表性土样约15-30g，放入铝盒中，称取铝盒和土样的总质量（m₁），准确至0.01g
  3. 将铝盒放入烘箱中，在105-110℃温度下烘干至恒重（一般需要8小时以上）
  4. 将烘干后的铝盒从烘箱中取出，放入干燥器中冷却至室温
  5. 称取铝盒和烘干土样的总质量（m₂），准确至0.01g
  6. 计算土的含水率：w = [(m₁ - m₂) / (m₂ - m₀)] × 100%

图4：比重瓶法测定土粒比重

4.3 比重试验（比重瓶法）

  1. 将比重瓶洗净、烘干，称取比重瓶的质量（m₀），准确至0.001g
  2. 取代表性土样约10-15g，放入比重瓶中，称取比重瓶和土样的总质量（m₁），准确至0.001g
  3. 向比重瓶中注入纯水至瓶的一半左右，然后放入真空抽气设备中抽气，排除土样中的空气
  4. 抽气后，向比重瓶中注满纯水，擦干瓶外水分，称取比重瓶、土样和水的总质量（m₂），准确至0.001g
  5. 倒出土样，洗净比重瓶，注满纯水，擦干瓶外水分，称取比重瓶和水的总质量（m₃），准确至0.001g
  6. 记录试验时的水温t℃，查得该温度下水的密度ρwt
  7. 计算土粒比重：Gₛ = (m₁ - m₀) / [(m₃ - m₀) - (m₂ - m₁)] × Gwt
  8. 式中：Gwt为t℃时水的比重，Gwt = ρwt / ρw4℃，ρw4℃为4℃时水的密度，取1g/cm³

六、注意事项

  1. 密度试验时，环刀应垂直下压，避免土样受到扰动
  2. 含水率试验时，土样应在105-110℃温度下烘干至恒重，避免温度过高导致有机质分解
  3. 比重试验时，应充分排除土样中的空气，确保试验结果准确
  4. 所有称量操作应迅速，避免水分蒸发影响试验精度
  5. 使用天平时，应轻拿轻放，避免损坏仪器
  6. 试验结束后，应及时清理试验设备，保持实验室整洁

七、思考题

  1. 土的密度、干密度和饱和密度有什么区别？它们在工程中的应用有什么不同？
  2. 为什么要进行土的含水率试验？含水率对土的工程性质有什么影响？
  3. 土粒比重的大小与哪些因素有关？不同类型土的比重一般在什么范围？
  4. 孔隙比和孔隙率有什么关系？它们如何反映土的密实程度？
  5. 饱和度的大小说明了什么问题？如何根据饱和度判断土的含水状态？
