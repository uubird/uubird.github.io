第4章 土的压缩性与地基沉降 - 习题

通过练习题掌握土的压缩性指标和地基沉降计算方法

习题说明

本章习题涵盖土的压缩性指标、地基最终沉降计算、地基沉降与时间关系等内容。通过这些习题，你将掌握地基沉降的计算方法和工程应用。

一、概念题

1. 什么是土的压缩性？土的压缩性指标有哪些？

参考答案

土的压缩性：土在压力作用下体积减小的特性称为土的压缩性。

主要压缩性指标：

  • 压缩系数a：土在侧限条件下，压力变化量与相应的孔隙比变化量的比值
  • 压缩指数C₁：e-logp曲线中，直线段的斜率
  • 压缩模量Eₛ：土在侧限条件下，竖向应力与竖向应变的比值
  • 体积压缩系数mᵥ：土在侧限条件下，单位压力变化引起的体积应变
  • 回弹指数Cᵣ：土在卸载过程中，e-logp曲线中直线段的斜率

2. 简述分层总和法计算地基最终沉降量的基本步骤。

参考答案

分层总和法计算地基最终沉降量的基本步骤：

  1. 选择沉降计算点的位置，计算基底压力和基底附加压力
  2. 确定地基压缩层厚度（一般取附加应力等于自重应力的20%处）
  3. 将压缩层厚度范围内的地基土分层，每层厚度一般不超过0.4倍基础宽度
  4. 计算各分层土的自重应力平均值和附加应力平均值
  5. 确定各分层土的压缩性指标（如压缩模量Eₛ）
  6. 计算各分层土的压缩量Δsᵢ = (σₛᵢ + σₚᵢ)/(2Eₛᵢ) × hᵢ
  7. 将各分层的压缩量相加，得到地基最终沉降量s = ΣΔsᵢ
  8. 根据工程经验，乘以沉降计算经验系数ψₛ，得到修正后的沉降量

3. 什么是土的固结？简述太沙基一维固结理论的基本假设。

参考答案

土的固结：饱和土在附加应力作用下，孔隙水逐渐排出，孔隙体积减小，土骨架逐渐承担荷载的过程称为土的固结。

太沙基一维固结理论的基本假设：

  • 土是均质、各向同性的饱和土
  • 土的压缩是一维的（竖向压缩）
  • 土的孔隙水渗流服从达西定律
  • 土颗粒和水是不可压缩的
  • 在固结过程中，土的渗透系数k和压缩系数a保持不变
  • 外荷载是一次瞬时施加的

4. 什么是先期固结压力？它对地基沉降计算有什么影响？

参考答案

先期固结压力：土在历史上曾经受到过的最大有效应力称为先期固结压力，用pₑ表示。

对地基沉降计算的影响：

  • 根据先期固结压力与当前有效应力的关系，土可分为正常固结土、超固结土和欠固结土
  • 不同固结状态的土具有不同的压缩特性，需要采用不同的压缩性指标
  • 超固结土的压缩性低于正常固结土，欠固结土的压缩性高于正常固结土
  • 在沉降计算中，需要根据土的固结状态选择合适的压缩曲线段
  • 对于超固结土，当附加应力小于超固结压力与当前有效应力之差时，应使用回弹曲线计算

二、计算题

1. 某土样的压缩试验数据如下：当压力p=100kPa时，孔隙比e=0.85；当p=200kPa时，孔隙比e=0.81；当p=300kPa时，孔隙比e=0.78；当p=400kPa时，孔隙比e=0.76。试计算该土样的压缩系数a₁-₂和压缩模量Eₛ₁-₂。

参考答案

解：

计算压缩系数a₁-₂：

a₁-₂ = (e₁ - e₂)/(p₂ - p₁) = (0.85 - 0.81)/(200 - 100) = 0.04/100 = 0.0004 MPa⁻¹

计算压缩模量Eₛ₁-₂：

Eₛ₁-₂ = (1 + e₁)/a₁-₂ = (1 + 0.85)/0.0004 = 1.85/0.0004 = 4625 kPa = 4.625 MPa

答案：压缩系数a₁-₂ = 0.0004 MPa⁻¹，压缩模量Eₛ₁-₂ = 4.625 MPa

2. 某矩形基础底面尺寸为3m×4m，埋深2m，地基土为粉质粘土，压缩模量Eₛ=5MPa，重度γ=18kN/m³。作用在基础顶面的竖向荷载为3000kN。试按分层总和法计算地基最终沉降量（取压缩层厚度为基础宽度的2倍）。

参考答案

解：

1. 计算基底压力和基底附加压力：

基底面积A = 3m × 4m = 12m²

基础及回填土自重G = 20×12×2 = 480 kN

基底压力p = (F + G)/A = (3000 + 480)/12 = 3480/12 = 290 kPa

基底附加压力p₀ = p - γ₀d = 290 - 18×2 = 290 - 36 = 254 kPa

2. 确定压缩层厚度：取基础宽度的2倍，即zₙ = 2×3 = 6m

3. 分层计算（每层厚度取1.5m）：

分层 深度范围(m) 层厚hᵢ(m) 自重应力σₛᵢ(kPa) 附加应力σₚᵢ(kPa) 平均应力(kPa) 压缩量Δsᵢ(mm)
1 2-3.5 1.5 (2×18 + 3.5×18)/2 = 49.5 254×0.8 = 203.2 (49.5 + 203.2)/2 = 126.35 (126.35/5000)×1500 = 37.9
2 3.5-5.0 1.5 (3.5×18 + 5×18)/2 = 76.5 254×0.4 = 101.6 (76.5 + 101.6)/2 = 89.05 (89.05/5000)×1500 = 26.7
3 5.0-6.5 1.5 (5×18 + 6.5×18)/2 = 103.5 254×0.2 = 50.8 (103.5 + 50.8)/2 = 77.15 (77.15/5000)×1500 = 23.1

4. 计算地基最终沉降量：

s = ΣΔsᵢ = 37.9 + 26.7 + 23.1 = 87.7 mm

答案：地基最终沉降量约为87.7 mm

3. 某饱和粘性土地基，厚度10m，初始孔隙比e₀=0.8，压缩系数a=0.0004 MPa⁻¹，渗透系数k=1×10⁻⁸ cm/s。地面上作用大面积均布荷载p=100kPa。试计算：(1) 地基的最终沉降量；(2) 加荷1年后的固结度；(3) 加荷1年后的沉降量。

参考答案

解：

(1) 计算地基的最终沉降量：

s = (a/(1 + e₀)) × p × H

其中：a = 0.0004 MPa⁻¹ = 0.0004×10⁻³ kPa⁻¹ = 4×10⁻⁷ kPa⁻¹

e₀ = 0.8，p = 100 kPa，H = 10 m = 10000 mm

s = (4×10⁻⁷/(1 + 0.8)) × 100 × 10000 = (4×10⁻⁷/1.8) × 1000000 = 0.000222 × 1000000 = 222 mm

(2) 计算加荷1年后的固结度：

固结系数Cᵥ = k(1 + e₀)/(aγw)

其中：k = 1×10⁻⁸ cm/s = 1×10⁻¹⁰ m/s

γw = 10 kN/m³

Cᵥ = 1×10⁻¹⁰×(1 + 0.8)/(4×10⁻⁷×10) = 1×10⁻¹⁰×1.8/(4×10⁻⁶) = 4.5×10⁻⁵ m²/s

时间因数Tᵥ = Cᵥt/H²

其中：t = 1年 = 365×24×3600 = 31536000 s

H = 10m（双面排水）或5m（单面排水，假设为单面排水）

假设为单面排水，H = 5m

Tᵥ = 4.5×10⁻⁵×31536000/(5²) = 4.5×10⁻⁵×31536000/25 = 56.765

对于单面排水，当Tᵥ > 0.1时，固结度U ≈ 1 - 0.81e^(-π²Tᵥ/4)

由于Tᵥ很大，U ≈ 100%

(3) 加荷1年后的沉降量：

sₜ = U×s = 100%×222 = 222 mm

答案：

(1) 地基的最终沉降量为222 mm

(2) 加荷1年后的固结度约为100%

(3) 加荷1年后的沉降量约为222 mm

三、综合分析题

1. 试分析影响地基沉降的主要因素，并说明在工程设计中如何减小地基沉降。

参考答案

影响地基沉降的主要因素：

  1. 地基土性质：
    • 土的压缩性：压缩系数、压缩指数等
    • 土的渗透性：影响固结速度
    • 土的应力历史：先期固结压力
  2. 荷载因素：
    • 荷载大小：直接影响附加应力
    • 荷载分布：均匀荷载与不均匀荷载
    • 荷载施加速率：影响土的固结过程
  3. 基础条件：
    • 基础形式：条形基础、独立基础、筏板基础等
    • 基础尺寸：影响附加应力分布
    • 基础埋深：影响基底附加压力
  4. 地质条件：
    • 地基土层分布：均匀性、厚度等
    • 地下水位：影响土的有效应力

减小地基沉降的措施：

  1. 地基处理：
    • 采用换填法、强夯法、挤密法等
    • 使用桩基础或复合地基
    • 进行预压处理（堆载预压、真空预压）
  2. 结构措施：
    • 增大基础底面积，减小基底压力
    • 采用刚度较大的基础形式（如筏板基础、箱形基础）
    • 设置沉降缝，避免不均匀沉降
  3. 施工措施：
    • 控制施工速度，避免快速加载
    • 合理安排施工顺序
    • 加强监测，及时调整施工方案
  4. 其他措施：
    • 控制地下水位
    • 选择合适的建筑场地
    • 预留沉降量

2. 某建筑物建成后，发现地基沉降超过了设计允许值。试分析可能的原因，并提出相应的处理方案。

参考答案

可能原因：

  1. 勘察资料不准确：
    • 地质勘察深度不足
    • 未发现软弱下卧层
    • 土的物理力学指标测试不准确
  2. 设计计算误差：
    • 未考虑实际荷载情况
    • 沉降计算方法选择不当
    • 沉降计算经验系数取值不合理
  3. 施工质量问题：
    • 地基处理未达到设计要求
    • 基础施工质量不合格
    • 超挖超填现象
  4. 使用条件变化：
    • 实际荷载超过设计荷载
    • 地下水位变化
    • 周边环境影响（如相邻建筑施工）

处理方案：

  1. 加固地基：
    • 采用注浆加固地基
    • 设置地基加固桩（如微型桩、静压桩等）
    • 进行地基托换处理
  2. 加强上部结构：
    • 增设圈梁、构造柱
    • 加固梁、板、柱等承重构件
    • 采用预应力技术
  3. 调整使用条件：
    • 限制建筑物使用荷载
    • 控制不均匀荷载
    • 设置沉降观测点，加强监测
  4. 应急处理：
    • 对于严重沉降，可能需要部分拆除重建
    • 设置临时支撑
    • 进行基础纠偏

四、工程实践题

1. 某高层建筑场地地基土层分布为：0-5m为杂填土，5-15m为淤泥质粘土（Eₛ=3MPa），15-25m为粉质粘土（Eₛ=8MPa），25m以下为基岩。建筑物高度80m，采用筏板基础，基础埋深6m。试设计一个地基沉降计算方案，并提出减小地基沉降的工程措施。

参考答案

地基沉降计算方案：

  1. 确定计算参数：
    • 收集建筑物荷载资料，计算基底压力
    • 进行详细的工程地质勘察，获取各层土的压缩性指标
    • 确定地基压缩层厚度（考虑应力扩散，计算至基岩顶面）
  2. 选择计算方法：
    • 采用分层总和法计算最终沉降量
    • 考虑应力历史，区分正常固结土和超固结土
    • 使用沉降计算经验系数进行修正
  3. 计算地基沉降与时间关系：
    • 采用太沙基一维固结理论计算沉降随时间的发展
    • 预测建筑物施工期间和使用期间的沉降
  4. 验证计算结果：
    • 与类似工程经验进行对比
    • 考虑相邻建筑物的影响

减小地基沉降的工程措施：

  1. 地基处理：
    • 采用桩基础，将荷载传递到深层粉质粘土层或基岩
    • 对于淤泥质粘土层，可考虑进行预压处理
    • 采用水泥土搅拌桩等复合地基
  2. 基础设计：
    • 增大筏板基础的刚度
    • 合理布置筏板厚度，加强整体性
    • 设置沉降后浇带
  3. 结构措施：
    • 优化上部结构布置，减小不均匀荷载
    • 增强上部结构的整体刚度
    • 设置足够的沉降缝
  4. 施工控制：
    • 采用分层施工，控制加载速率
    • 加强沉降监测，实行信息化施工
    • 合理安排施工顺序，避免相互影响

2. 某工业厂房采用柱下独立基础，地基为软弱粘性土，压缩模量Eₛ=2.5MPa，厚度12m。在使用过程中，发现厂房不均匀沉降，柱顶倾斜超过允许值。试分析事故原因，并提出加固处理方案。

参考答案

事故原因分析：

  1. 地基因素：
    • 软弱粘性土压缩性高，承载力低
    • 地基土不均匀，导致不均匀沉降
    • 可能存在软弱下卧层未被发现
  2. 设计因素：
    • 独立基础底面积不足，基底压力过大
    • 未考虑厂房荷载的不均匀分布
    • 沉降计算不准确，未预留足够的沉降量
  3. 施工因素：
    • 基础施工质量不合格
    • 地基土受到扰动
    • 未按设计要求进行地基处理
  4. 使用因素：
    • 实际荷载超过设计荷载
    • 设备布置不均匀
    • 周边环境变化（如地下水位下降）

加固处理方案：

  1. 基础加固：
    • 采用基础扩大托换，增加基础底面积
    • 设置基础联系梁，增强整体性
    • 对基础进行化学加固（如压力注浆）
  2. 地基加固：
    • 在独立基础下设置微型桩或静压桩
    • 采用高压喷射注浆形成桩体复合地基
    • 对地基进行深层搅拌加固
  3. 上部结构加固：
    • 加固柱、梁等承重构件
    • 增设支撑系统，减小柱顶位移
    • 进行柱顶纠偏处理
  4. 监测与控制：
    • 设置沉降观测点，进行长期监测
    • 控制使用荷载，避免超载
    • 调整设备布置，减小不均匀荷载
